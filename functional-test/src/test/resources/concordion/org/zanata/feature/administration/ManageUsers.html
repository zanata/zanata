Management of User Details


Changing a User's Login Details

As an Administrator, I want to change a User's username and/or password.

To change a user's username or password, navigate to the User Management page by

  1. Opening the More menu and selecting Administration
  2. Selecting Manage Users

and navigate to user 'admin' and press Edit.

Change the Username, Password and Confirm Password fields to the desired values and press Save.
In this example, we change admin to administrator and the password to password. press the Save button and the new user details will be displayed.
